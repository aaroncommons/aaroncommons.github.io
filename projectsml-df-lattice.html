Adjoint Optimisation and Machine Learning-Based Tuning of the GEKO Turbulence Model

Context

This MSc thesis project investigated limitations in current RANS turbulence modelling approaches used in aircraft aerodynamic design, with a focus on improving prediction accuracy using adjoint optimisation and machine learning techniques.

Objectives

  • Evaluate baseline performance of the GEKO turbulence model
  • Apply adjoint optimisation techniques to improve aerodynamic prediction accuracy
  • Investigate the use of machine learning for turbulence model tuning

Methodology

  • CFD simulations performed using ANSYS Fluent
  • Airframe geometries analysed under representative flow conditions
  • Adjoint-based sensitivity analysis used to guide optimisation
  • Python used for data processing, automation, and ML workflow

Results

The optimised model demonstrated improved agreement with reference aerodynamic performance metrics compared to baseline RANS predictions, highlighting the potential for targeted turbulence model tuning in early-stage aircraft design.

Engineering Insight

This project demonstrated how advanced numerical methods and optimisation techniques can be directly applied to address practical limitations in aircraft CFD prediction, bridging the gap between research methods and real-world engineering design workflows.

Tools Used

  • ANSYS Fluent
  • Python
  • Machine Learning (TensorFlow / PyTorch)
